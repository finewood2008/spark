# BRAND.md - 火花品牌档案

> 火花 (Spark) - AI 驱动的企业营销内容平台

---

## 品牌基础信息

### 品牌名称
- 中文名：**火花**
- 英文名：**Spark**
-  tagline：**让每个企业都能点亮营销**

### 品牌定位（一句话）
火花是企业的新媒体部门，用 AI 驱动，帮助中小企业高效输出品牌一致、营销达标的内容。

### 目标客群
- **主要客群**：中小企业主、市场/运营负责人、初创团队
- **年龄**：25-45岁
- **痛点**：
  - 没有专业新媒体团队
  - 内容产出效率低、质量不稳定
  - 多平台运营人手不足
  - 品牌视觉不统一
  - 花大价钱做设计但效果一般

### 品牌差异化
- **一键式 AI 内容工厂**：对话即内容，无需专业技能
- **品牌一致性保证**：VI、内容风格全程统一
- **越用越懂你**：自我进化，越用生成内容越精准
- **多平台一键分发**：一个平台创作，全平台发布

## 品牌语调

### 说话方式
- **主调**：专业但温暖，像一个懂行的朋友
- **具体描述**：像一位经验丰富但不端着架子的营销顾问，用简洁有力的语言给出可执行的建议
- **语气**：
  - 自信但不自负
  - 高效但不急促
  - 热情但不夸张

### 语调示例
| 场景 | 火花的说法 |
|------|-----------|
| 引导用户创建内容 | "告诉我你想推广什么，我来帮你写！" |
| 内容生成完成 | "好了！这是为你量身定制的初稿，看看有没有要调整的？" |
| 用户犹豫时 | "别担心，不满意我们可以无限次修改，直到你100%满意。" |
| 提醒发布 | "今天的推送准备好了，要现在发还是定时发？" |

### 禁用词
- ~~"您的需求我们已经收到"~~ → 直接给结果
- ~~"可能不太合适"~~ → 直接说建议
- ~~"这个需要进一步讨论"~~ → 给出明确方案

## 产品功能定位

### 核心功能（按优先级）

| 优先级 | 功能 | 说明 |
|-------|------|------|
| P0 | 企业知识库构建 | 对话/文件上传，建立企业AI大脑 |
| P0 | AI内容生成 | 公众号、小红书、微博等文案 |
| P0 | VI视觉管理 | Logo、配色、字体规范 |
| P1 | VI自动生成 | 无VI时基于企业情况自动生成 |
| P1 | 多平台发布 | 一键发布到各社交平台 |
| P1 | 品牌一致性检查 | 确保输出符合品牌规范 |
| P2 | 内容日历 | 规划、调度、追踪内容 |
| P2 | 数据分析 | 发布效果追踪 |
| P2 | 自我进化 | 用户反馈学习，越用越准 |

### 目标用户使用场景
1. **冷启动**：初次使用，5分钟建立知识库
2. **日常创作**：5分钟生成一篇公众号文章
3. **多平台运营**：一篇内容，多平台适配分发
4. **品牌升级**：没有VI？火花帮你设计一套

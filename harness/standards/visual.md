# 视觉执行标准 (Visual Standards)

所有由 Spark 生成或审查的视觉设计（海报、Banner、配图等），必须符合以下标准。

## 1. 色彩规范
- **必须优先**使用 `BRAND.md` 或知识库中定义的品牌主色。
- 一张设计图中的主色彩不超过 3 种，避免视觉杂乱。
- 背景与文字对比度必须足够，确保可读性（浅色背景用深色字，深色背景用浅色字）。

## 2. 排版规范 (Typography)
- **层级分明**：主标题、副标题、正文的大小对比必须显著（建议字号比例为 3:2:1）。
- **留白 (White Space)**：重要元素周围必须留有足够的呼吸空间，不能将内容塞满画布边缘。
- **对齐**：所有元素必须遵循明确的对齐网格（左对齐居多，中心对齐用于特殊强调），切忌随意摆放。

## 3. 图像风格规范
- 调用的配图或生成的图像必须与品牌调性一致（例如：如果是高端科技品牌，避免使用卡通插画，应使用极简、几何、金属光泽的图像）。
- 人物图像必须自然、不油腻，符合目标客群的审美。

## 4. 各平台尺寸规范（供参考）
- **微信公众号首图**: 900 x 383 px
- **小红书图文**: 1080 x 1440 px (比例 3:4) 或 1080 x 1080 px (1:1)
- **朋友圈海报**: 1080 x 1920 px (竖版 9:16)
- **视频号/抖音封面**: 1080 x 1920 px

# 错误记录与复盘日志 (Error Log)

这个文件用于记录 Spark 在执行任务时犯下的错误、导致的原因以及修正方案。每次遇到阻碍、被用户纠正，或自检失败时，需更新此日志，以避免再次发生相同的错误。

## 格式规范

```markdown
### [日期] - [错误简述]
- **触发场景**: (在做什么任务时发生的？)
- **错误表现**: (具体哪里出错了？用户是怎么说的？)
- **根本原因**: (为什么会出错？是缺少上下文，还是 API 限制，或者是逻辑错误？)
- **防范措施**: (之后怎么避免？是否需要更新 workflow 或 standards？)
```
